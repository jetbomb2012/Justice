# 案件當事人: 彭恢華 
# ( 授權立法、司法人員引用不法事證光碟文件 )
# 案件為95年6月10日九合一選舉舞弊案
# 一、事經桃檢署、高檢署、交付審判桃院三法官、民事法院法
#  官等皆隱匿王宗堂身分 ( 縣長代理人 ) 違反專屬管轄並為不法
#  確定
# 二、經此不法事由，並確認王宗堂身分、職務行為，向高等法
#  院、最高法院、最高行政法院依法聲請移轉管轄
# ，但未有移轉管轄作為，至今仍無法院管轄逾18年，
# 有違大法官解釋「適時審理請求之權」。
# 三、故依國家根本大法之客觀、外觀明文，
#  認定「本案有違憲之情」：
# 憲法第16條「人民有請願、訴願及訴訟之權」、
#    憲法第132條「選舉應嚴禁威脅利誘。
# 選舉訴訟，由法院審判之」、
#    大法官解釋「適時審理請求之權」為憲法當然之理。
# 四、故本件司法人員違反法令致「選舉案未經法院審判、
# 選舉案件違反專屬法院管轄」，
# 進而剝奪人民訴訟權，並無限期拖案有違適時審理請求之權，
# 「違反憲法132條、16條故違憲至為明確」。
# 五、其餘就本案相關之違法案件，皆無從審理，
# 致犯罪漏洞大開。
