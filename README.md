# 聲明事項補充說明: ( 承前聲明法院指定移轉管轄案，依法准、駁辦理 )
# 一、選舉制度非為一人、一時而設，況憲法、選罷法明文專屬法院合議，今檢察機關違反、侵官並實質違法不起訴處分須撤銷，不當認定專屬管轄之情( 當然違反法律事項 ) 。況隱匿違法身分、涉案事實、侵官生事，其藉故積極違法管轄，侵害法院專屬法院合議權力，情節重大。
# 二 . 桃檢96年選偵6號: (自始、當然、確定違反專屬管轄而違法無效)
# 案情:王宗堂職司選務監督公務人員(舊法)，時兼國民黨常委職於95.3.23日「不法無因管理」，涉入干擾控制他人選舉提名會議, 事後有國民黨有權代理人傅忠雄更正而不理，致生他人選舉期間，特殊身分權 (人格權) 受損而連帶財務受損，二事具相當因果關係。
# 三.今檢察機關認(新法不違法)隱匿王宗堂職司選務監督身分，行為事實，未移專屬管轄三人合議法院，「係重大明顯瑕疵」，此行政高等法院確認訴訟程序中其回覆文中，明白確認身分、事實。國賠與選舉為二類專屬管轄事件，不可混而遁入！必先釐清基礎事實以符專屬意旨，始生實益。(況王員為舊法公務人員明確，檢察機關蓄意違法明確。)
# 四. 三人以上法院選舉案合議後始生有法所不許案由，如:行政法、刑法、民法、行政中立法、公務人員選罷法、公務人員服務法，其它不及備。
